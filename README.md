GradeBook Analyzer

A simple Python CLI tool that helps analyze student marks using manual input or a CSV file.

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

# Project Overview


This mini project automates basic gradebook tasks. It reads student names and marks, calculates statistics like average and median, assigns grades using if and elif, separates passed and failed students using list comprehensions, and finally prints everything in a clean results table.

It also allows exporting the final results to a CSV file.

This project was created as part of the Programming for Problem Solving using Python course.

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

# Features


- Manual data entry for student names and marks


- Load marks from a CSV file


- Computes statistics


-Average


-Median


-Highest score


-Lowest score


- Grade assignment (A to F)
  

- Grade distribution summary


- Pass and fail filtering using list comprehension


- Neat formatted results table


- Export results to a CSV file


- Repeat analysis using a simple CLI menu
